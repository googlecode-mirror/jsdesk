################
# Requirements #
################

Web server running PHP/MySQL
Ext JS library (www.extjs.com)

################
# Installation #
################

1. Edit jsDesk/inConfig.php to point to your mysql host and enter your username, password, database for the server. Change the rest of the variables to fit your server.

2. Import jsDesk/sql/*.sql into your mysql database

3. Edit jsDesk/inHeader.php to point to a copy of the Ext JS library 

4. Upload the jsDesk/ (root) folder and it's contents to your webserver (also upload extjs wherever you like)

5. open http://[YOUR_WEB_SERVER]/jsDesk or whatever directory you put it in

6. rock out \m/
